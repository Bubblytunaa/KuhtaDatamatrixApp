Андроид приложение автором которго является Кухта А.Л.
Приложение позволяет генерировать визуальный код DataMatrix из cимволов ASCII вводимых в строку пользователем.
Для установки приложения скачайте файл расположенный по адресу DataMatrix-kuhta\app\release\app-release.apk
Приложение написано для ОС Анроид 11.0. Минимальная ОС необходимая для запуска - 4.1.
